# Axibase Collector Requirements

# Hardware

  • Processors – 1 CPU, 2+ GHz
  • Memory – 1GB (512MB for JVM)

# Architecture

  • x86_64 (64 bit)

# Operating Systems

  • Ubuntu 14+
  • Debian 6+
  • RedHat Enterprise Linux 6+
  • CentOS 6+
  • SUSE Linux Enterprise Server 11.2+

# Java

  • JDK 1.7
