# Using axibase_nginx_plus_collector Script for Monitoring NGINX Plus Servers

To collect information from your NGINX Plus servers, download the axibase_nginx_plus_collector.py and axibase_npc_core.py scripts and run the axibase_nginx_plus_collector.py script with the appropriate options. The help message can be printed after running:

python axibase_nginx_plus_collector.py --help

Supported options:

OptionExtended optionDescriptionDefault
-a--atsd-urlURL of target ATSDtcp://localhost:8081
-i--itemsSpace separated NGINX Plus servers item listhttp://demo.nginx.com
-q--quietFlag indicating the script will not generate outputFalse
